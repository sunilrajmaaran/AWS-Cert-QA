Amazon Athena is a serverless query service that lets you run SQL queries directly on data stored in Amazon S3.

You don’t need to set up servers, databases, or ETL pipelines — Athena queries the data “as is.”

It supports structured, semi-structured, and unstructured formats like JSON, CSV, Parquet, ORC, Avro.
